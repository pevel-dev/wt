# Telegram Weight Tracker Bot

Telegram бот для трекинга похудения. Написан на связке Django (как ORM + Админка) + aiogram 3.x + arq для фоновых задач.

## Стек технологий
- Python 3.12
- Django 5.2
- aiogram 3.x
- arq (Redis Task Queue)
- PostgreSQL
- Redis

## Как развернуть (в том числе в Coolify)

### Предварительные требования
Для запуска нужен только Docker и Docker Compose.

### Переменные окружения (`.env`)
Минимально необходимый набор переменных:

```env
# Телеграм
BOT_TOKEN=123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11

# Django
DJANGO_SECRET_KEY=your-super-secret-key
DEBUG=False

# База данных и Redis (для docker-compose)
DATABASE_URL=postgres://wt_user:wt_password@db:5432/weight_tracker
REDIS_URL=redis://redis:6379/0
```

### Запуск
```bash
# 1. Сборка и запуск контейнеров
docker-compose up -d --build
```

При старте сервиса `web` автоматически выполняется `python manage.py migrate --noinput`, затем поднимается Django (админка на порту 8000).

```bash
# 2. Создание суперпользователя (для доступа к админке)
docker-compose exec web python manage.py createsuperuser
```

Админка будет доступна по адресу `http://<your-ip>:8000/admin`.

## Команды бота
- `/start` - Регистрация, установка текущего и целевого веса
- `/weight` - Обновление веса
- `/measurements` - Внесение замеров (грудь, талия, бедра)
- `/add_habit` - Создание новой привычки для ежедневного трекинга
- `/habits` - Список привычек
- `/today` - Вызов меню с кнопками для отметки привычек на сегодня
- `/progress` - Твой прогресс (этапы, дедлайн, статус)
- `/stats` - Статистика выполнения привычек

## Логика работы
- **Цель:** Бот строит реалистичный план похудения исходя из нормальной потери **1% веса в неделю**. На основе этого рассчитывается дата-дедлайн.
- **Этапы:** Если вес больше 80кг, путь разбивается на промежуточные цели по 5кг. Когда вес становится меньше 80кг, шаги уменьшаются до 2кг.
- **Привычки:** Если в конце дня пользователь не отметил все свои привычки как выполненные, дата дедлайна сдвигается на 1 день вперед.
- **Уведомления:** 
  - Каждый день в 20:00 бот присылает inline-клавиатуру с привычками.
  - Каждое воскресенье в 10:00 напоминает занести вес и замеры.
  - Каждую ночь (в 01:00) бот проверяет, закрыл ли пользователь привычки за вчера, и при необходимости сдвигает дедлайн.
